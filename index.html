Here is my favorite Python script that I have written. It's purpose was to convert pixel data as ASCII and convert it into an actual image. I wrote this script as to find a flag in a cybersecurity CTF challenge. 

                        from PIL import image
                        size = (17*17,37)

                        with open('secret_data.txt',"r") as f:
                        beep = f.readlines()
                        supercoolmegalist = b""

                        for x in range(len(beep)):
                            for lol in beep[x].split("-"):
                                supercoolmegalist += ((int(lol)).to_bytes(1,'big'))
                        print(supercoolmegalist)

                        img = Image.frombytes("RGB",size,supercoolmegalist)

                        img.save('flag.png')
